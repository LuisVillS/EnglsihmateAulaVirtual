# CRM Plan M3

## Scope

- CRM Kanban, lead detail workflows, and operator-focused UI.

## Current Milestone Work

- Add dynamic pipeline value calculation to `/admin/crm/kanban`.
- Add per-stage edit controls from the Kanban header menu.
- Add rename-safe stage fields: `system_key` and `display_name`.
- Add stage-level automation settings for initial and stagnancy follow-up emails.

## Exit Criteria

- Kanban uses fixed pipeline math with the unified course price.
- Stage editing from Kanban persists safely.
- Stage automations continue to use stable identifiers after a display-name rename.
